2022/01/22

マジでやることがないので日記をつけ始めることにする。 静的サイトジェネレーターのHugoで作ってgithub pagesでホストする。

知らん人の興味ない投稿が勝手に表示されたりいらん広告勝手に表示されるTwitterとかいうやつやめてみんなで個人サイトを開設しないか？相互リンク募集中です。

ブログの構築

簡単に Hugo サイトを GitHub Pages で（ 自動ビルドして ）公開する方法

これを参考に自動でサイトをビルドしてくれるようにしようと思う。

と思っていたのだが、めんどくさくなったので記事を書く→ビルド→githubにプッシュまでを一つのコマンドにまとめる程度にしておく。とりあえずはメインPCからしか更新できなくなるけど、まあこれでいい。どうせそのうち飽きて更新しなくなるので。

方針変更

本当はWSL上でやるつもりだったんだけど、なんかWSL上のhugoだとプレビューがうまく行かなかったのでWindowsにHugo入れてやることに。

Hugo 静的サイトジェネレーターによるサイト構築と公開

これを参考に環境構築。 Sourcetreeを使ってgithubにpushしてみたけど、なんかpagesでうまく表示されん。謎。

↓

解決。

【Hugo】hugo serverで動くのに、ビルド後は正しく動かないときの対応【582日目】

このせいだった。この場合は、config.tomlのbaseURLを

baseurl = "https://tkmtn.github.io/blog/"

とすることで解決した。

これでpushしても治らないように見える場合があるけど、github側でのキャッシュが更新されるまで待つと治った。

ビルドとpushの手順

  1. コンテンツをつくる
  2. hugo -Dでビルド
  3. Sourcetreeでcommit, push

正直めんどい

ビルドからpushまで自動化したい、気が向いたら
