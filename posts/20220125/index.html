睡眠状況をツイートする

otokura1922は私のTwitterアカウントです。

でもタイムラインは見てないし私が手動でツイートすることもありません。ログインしてる人が何人かいるのでその人がツイートすることがあります。

*Auto*

と頭についているものはTwitter APIを使用した自動ツイートです。

今回はこれの中身について紹介します。私がこれを作るときは全部ググった記事をパクりながら作りました。プログラムの知識なんてほとんど必要無いですし、簡単なので皆さんも是非作ってみてくださいね。

よりよい方法をお持ちの方、他の方法で同じようなことをやってる方がいればぜひコメントしてください。

Cloud Firestoreに睡眠時間を記録するDBを作る

Cloud Firestore

次に使うCloud Functionsは変数を長時間保持できないので、どこかに就寝時間を保存する必要があります。私はGCPのデータベースであるFirestoreを使いました。

上の画像の青色のコレクションを開始とかドキュメントを追加とかから今回使うためのデータベースを追加します。データのフィールドもこの段階で作っておきましょう。

Cloud Functionsで睡眠監視を受け取る関数を作る

Google Cloud Functions(GCF)はいわゆるFaaS(Function as a Service)と呼ばれるサービスで、かんたんな設定でアプリケーションを開発することができます。

GCFに下のようなスクリプトを登録して、HTTPで呼び出せるように設定しておきます。これはデフォルトで入っているHello Worldスクリプトを改造して作りました。

ランタイムはpython3.7ですが、バージョンは何でもいいと思います。 TwitterのトークンなどはSecret Managerに入れて環境変数にしています。

import datetime
from google.cloud import firestore
import json
import tweepy
import os

def sleep_watch(request):
    """Responds to any HTTP request.
    Args:
        request (flask.Request): HTTP request object.
    Returns:
        The response text or any set of values that can be turned into a
        Response object using
        `make_response <http://flask.pocoo.org/docs/1.0/api/#flask.Flask.make_response>`.
    """
    request_json = request.get_json()
    if request.method == 'POST':
        if request_json and 'content' in request_json:
            if request_json['content'] == 'fell_asleep':
                aslp()
                tweet('*Auto*\nおやすみなさい')
                return 'fell asleep'
            elif request_json['content'] == 'wake_up':
                # 最新の就寝時刻から現在時刻を引いて、全秒数から時間、分を計算する。
                slptime = wake()
                sec = slptime.total_seconds()
                hours, amari = divmod(sec, 3600)
                minutes, secs = divmod(amari, 60)
                slptime_formated = f'{int(hours)}時間{int(minutes)}分'
                tweet(f'*Auto*\nおはようございます。\n睡眠時間：{slptime_formated}')
                return slptime_formated

# Firestore
db = firestore.Client(project="プロジェクトID")
doc_ref = db.collection('sleepwatch').document('time')
def aslp():
    doc_ref.set({'fell_asleep_time':datetime.datetime.utcnow()})

def wake():
    doc = doc_ref.get().to_dict()['fell_asleep_time']
    now = datetime.datetime.utcnow().replace(tzinfo=None)
    slptime = now - doc.replace(tzinfo=None)
    return slptime

# Twitter
tw_auth = tweepy.OAuthHandler(os.getenv('TW_API_KEY'), os.getenv('TW_API_SECRET'))
tw_auth.set_access_token(os.getenv('TW_ACCESS_TOKEN'), os.getenv('TW_ACCESS_TOKEN_SECRET'))
api = tweepy.API(tw_auth)
def tweet(content):
    try:
        api.update_status(content)
    except:
        # 連続で同じ内容を投稿できないので、エラーが帰ってきたら現在時刻（UTC）を末尾に付加してツイートする。
        api.update_status(f"{content}\n{datetime.datetime.now()}")
